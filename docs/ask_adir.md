# רשימת "לשאול את אדיר" — פריטים לאימות / השלמה

נוצר אוטומטית מקובצי ה-raw. שני חלקים: מבחנים סרוקים להשלמה, ושאלות שסומנו לאימות.

## 1) מבחנים סרוקים — חולצו והוכנסו לבנק (לאימות מול מפתח אם קיים)

שלושת המבחנים הסרוקים נקראו חזותית מהתמונות והוכנסו לבנק. אין להם קובץ פתרונות, ולכן התשובות **נגזרו** (מסומנות 'לא רשמי') — שאלותיהם המסומנות מופיעות בסעיף 2. אם יש לך מפתח פתרונות עבורם, שלח ואהפוך את תשובותיהם לרשמיות:
